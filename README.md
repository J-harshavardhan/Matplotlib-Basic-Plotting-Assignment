# 📊 Matplotlib Data Visualization Assignment

## 📌 Project Overview
This project demonstrates the use of **Matplotlib**, a popular Python library for data visualization.  
The goal of this assignment is to analyze student performance data and create different types of visualizations to understand patterns, trends, and distributions.

Using the provided datasets, several plots were created to visualize relationships between study habits, subject performance, score distributions, and student activities.

---

## 📂 Datasets

### 1️⃣ Student Study Hours vs Marks
Shows the relationship between the number of hours students study and the marks they obtain.

### 2️⃣ Student Performance in Subjects
Contains student scores in **Mathematics** and **Science**.

### 3️⃣ Marks Distribution
Includes marks of **50 students** used to analyze the distribution of scores.

### 4️⃣ Student Activities Distribution
Shows how students participate in extracurricular activities such as Sports, Music, Arts, Drama, and Debate.

---

## 📊 Visualizations Implemented

### 🔹 Line Plot
Used to visualize the trend between **study hours and marks obtained**.

### 🔹 Scatter Plot
Displays individual data points to analyze the relationship between study hours and marks.

### 🔹 Grouped Bar Chart
Compares **Mathematics and Science scores** across different students.

### 🔹 Histogram
Shows the **distribution of marks** among 50 students using bins.

### 🔹Pie Chart
Represents the **percentage distribution of student participation in activities**.
